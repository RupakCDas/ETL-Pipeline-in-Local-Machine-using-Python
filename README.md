# ETL-Pipeline-in-Local-Machine-using-Python
I just built a simple ETL Pipeline using Python. Which extracts data from Crypto website API then transforms and load into MySQL database. I used a for loop that works as an automated orchestrator. Which runs tasks after a particular time.
